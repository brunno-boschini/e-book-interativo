# e-book-interativo

# Projeto Final — Desenvolvimento Front-End

Este repositório contém o site interativo criado como entrega final da disciplina Desenvolvimento Front-End — Universidade Católica de Brasília.

## Tecnologias
- HTML5
- CSS3 (Mobile-first)
- JavaScript

## Funcionalidades JS
- Smooth scroll
- E-book interativo com acordeão
- Quiz
- Validação de formulário

## Seções
- Landing Page
- Sobre
- E-book
- Galeria
- Vídeo multimídia
- Contato

## Acessibilidade
- Navegação por teclado
- Texto alternativo
- Labels em formulários

## Hospedagem
Recomendado: GitHub Pages.
